# Prophesy
[![DOI](https://zenodo.org/badge/1179792781.svg)](https://doi.org/10.5281/zenodo.18986525)

Prophesy is an Excel add-in for Cross-Impact Balances (CIB) analysis with improvements to succession analysis for handling ties, parallel processes, incremental vs. radical shifts, and auxiliary variables for complex causation. 

## Features
- Improved succession analysis
- Scenario analysis
- Perturbation analysis
- Pathway analysis
- Sensitivity analysis
- Discrete Maximin designs
- Descriptive statistics and diagnostics

## Installation
1. Download Prophesy.xlam.
2. Open Excel. 
3. Go to the Developer tab. If the Developer tab is not visible, go to File > Options > Customize Ribbon and check Developer, then click OK. 
4. Click Excel Add-ins. 
5. Click Browse. 
6. Find Prophesy.xlam, select it, and click OK. 
7. Click OK in the Add-ins window.
